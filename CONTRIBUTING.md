### üí° How You Can Help

You don't have to write code to contribute! Here are a few ways you can help:

* **Share your feedback:** specific suggestions on how to improve the app are incredibly helpful.
* **Documentation:** clear documentation is the backbone of any good project. If you see a typo or a confusing section, please fix it!
* **Design:** we want Demos to be accessible and beautiful. UI/UX contributions are highly valued.
* **Research:** help us find the best blockchain solutions or open-source datasets for community analysis.
* **Code:** of course, pull requests for bug fixes and new features are always welcome.

### üöÄ Getting Started with Code

1.  **Fork the repository** to your own GitHub account.
2.  **Clone the project** to your machine.
3.  **Create a branch** for your fix or feature (`git checkout -b my-amazing-feature`).
4.  **Commit your changes** (`git commit -m 'Add some amazing feature'`).
5.  **Push to the branch** (`git push origin my-amazing-feature`).
6.  **Open a Pull Request** and tell us about your changes!
